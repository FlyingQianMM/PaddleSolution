## 任务描述

  目标检测的任务是给定一张图片，让计算机找出其中所有目标的位置，并给出每个目标的具体类别。通常会用一个包围框来紧凑地框出物体的全部范围，以此表示目标的位置。用户可以使用YOLO V3来完成此项任务。
  
  实例分割的任务是在目标检测的基础上，找出各包围框内属于目标的像素点。用户可以使用Mask R-CNN来完成此项任务。
